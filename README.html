JSBridge协议文档

基本协议

1.JS调用Native

JS通过特定方式传递此json字符串到客户端 JSON格式 { "method":"methodname", //调用方法名称（必须有） "callback":"callback name", //设置回调入口(可以没有) "params":{ //调用所带的参数（可以没有） "param1":value1, "param2":value2 }, "extra":"optional" //未来预留的字段（可以没有） }

调用方法

1,通知ios消息的方式

iosH5_.H5CallNative(data); // data 为json字符串

2,通知安卓消息的方式

window.prompt(data); // data 为json字符串

2. Native回调JS

    {
        "errcode": 0        //0表示成功，  其他为错误码，见后面
        "errmsg": "xxx"     //错误信息，可以为空
        "data":{            //调用所带的参数
            "param1":value1,
            "param2":value2
        },
        "extra":"optional"  //未来预留的字段
    }

客户端通过 loadUrl("javascript:callback({callback_JSON})")的方式回调
